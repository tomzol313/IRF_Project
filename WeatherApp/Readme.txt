A program egy webszolgáltatásra épül, amellyel megjeleníthetőek az aktuális időjárási adatok és az előrejelzések is. A felhasználó elsősorban kiválaszthatja, hogy melyik amerikai város értékeire kíváncsi. A program többféleképp tudja majd megjeleníteni az adatokat. A felhasználó ezekre is rászűrhet majd, vagy választhat egy alapértelmezett megjelenítési módot. A kinyert adatokat le is töltheti egy CSV fájlba. A program az adott időjáráshoz illő képet fog majd megjeleníteni a graphics osztály segíségével.
